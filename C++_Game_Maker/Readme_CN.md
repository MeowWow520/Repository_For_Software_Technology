## 使用的图形库

本仓库使用[EasyX](https://easyx.cn/)图形库

>
>**EasyX Graphics Library** 是针对 Visual C++ 的免费绘图库，支持 VC6.0 ~ VC2022，简单易用，学习成本极低，应用领域广泛。目前已有许多大学将 EasyX 应用在教学当中。
>

**链接**

[EasyX 官方网站](https://easyx.cn/)： https://easyx.cn/
