## 这是什么？
### 简介
厌恶人类身体？那就做一个喵星人！耕种、浇水、收获，体验最纯粹的田园乐趣。从零开始，打造属于你的丰收乐园！🌾✨
### 玩法说明
使用 **W**、**A**、**S**、**D** 控制喵喵移动，鼠标 **右键** 开垦，鼠标 **左键** 选择作物和种植在耕地上。不同的作物，拥有不同的价值和不同的种植时间。
