## 文件命名规则

**文件夹**：

文件夹的命名方式有三种，分别为：
1. `Word_Word`  
2. `Word-Word`  
3. `Word Word`

命名时循环使用这三种命名方式。

**文件**：

文件的命名遵循蛇形命名法：使用 `_` 代替“空格”，且除专有名词以外的词语全部小写(特指英文版)
